Afronta el mundo actual y los conocimientos para forjar el futuro, permitiendo el desarrollo y la creación de nuevas aplicaciones o sistemas que puedan ser de ayuda para una empresa y la sociedad en general.

Desarrollo de Software

Desarrolla programas a través de diferentes lenguajes de programación.

Ingeniería de Software
